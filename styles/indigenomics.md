# Indigenomics Style Guide

## Overview

The `indigenomics` style is the visual language of the Indigenomics Institute — bridging ancestral wisdom with emergent technology. Mythic realism meets cyberpunk; cosmic yet grounded. Sovereignty, reciprocity, and emergence are the organizing principles. Humans and land are in the loop.

**Critical governance note:** This style guide is a *prompting aid*, not permission to invent or use sacred iconography. Nation-specific symbols (specific eagles, drums, salmon, longhouses, regalia) require appropriate Indigenous governance and consent. Use abstract geometry, generalized land-based metaphors, and the palette/motif vocabulary below — leave culturally specific references for human review and community co-design. When in doubt, ask an Indigenomics Institute contact before publishing.

## Color Palette

| Role | Colors |
|------|--------|
| Sky / spirit | Deep navy, indigo, midnight blue |
| Earth / ancestry | Burnt sienna, cedar brown, ochre |
| Tech / flow | Electric cyan, teal glow, copper, soft gold |
| Data overlay | White smoke, light grid blue, warm neon accents |

Glow (used sparingly): activated technology, data pulse, or **relationship** — not decoration.

## Key Motifs

- **Braided materials** — sweetgrass, cedar, copper wire interlaced with fiber-optic threads and circuit traces; the braiding itself is the metaphor (integration, not replacement)
- **Radial and circular flows** — council arrangements, circular reciprocity diagrams; avoid purely hierarchical layouts
- **Blueprint + organic overlay** — technical grid structure with organic growth patterns on top
- **River flow paths** — values → design → capital → outcomes, expressed as flowing water, not flowchart arrows
- **Geo-mapping with glowing nodes** — territory, community, and enterprise visibility

## Geometry

Radial symmetry, circular flows, blueprint grids with organic overlays, data streams and ripples, isometric scenes mixed with simple pictographs. Never purely top-down hierarchical.

## Typography

- Titles: all-caps serif or slab (gravitas, not decorative)
- Body: clean sans
- Minimize text baked into images — prefer overlays for section titles only

## Diagram Archetypes

1. **Blueprint fusion** — traditional structure + community circle + smart city (infrastructure + values in one frame)
2. **Living AI brain** — organic brain + circuitry (value-aligned processing, not robotic AI)
3. **Geo-mapping layer** — territory map with glowing nodes (visibility and sovereignty framing)
4. **Flow river path** — values → design → capital → outcomes
5. **Subverted race metaphor** — lead with Indigenous youth / futures; avoid "catch-up" framing entirely

## Data Representation

Prefer **relational** charts (flows, cycles, reciprocity diagrams) over transactional bar charts. When shaping data into meaningful forms (salmon, cedar, mountain silhouette), get community review before publishing.

## Tone

Mythic realism. Cosmic yet grounded. Humans-in-the-loop and land-based intelligence. Not dystopian, not techno-utopian — sovereignty-forward, futures-oriented, pragmatic about AI as a tool not a savior.

## When to Use

- Indigenomics Institute presentations and reports
- BC+AI content centering Indigenous economic sovereignty
- AI-and-land / AI-and-governance educational diagrams
- Partnership materials where Indigenous leadership is centered

## When NOT to Use

- Generic "multicultural" content (use `kk` or `bcai` instead)
- Anything requiring culturally specific symbols without governance review
- Contexts that frame Indigenous communities as catching up rather than leading

## Example CLI Commands

```bash
# Blueprint fusion — infrastructure + values
python generate.py -p "Traditional longhouse silhouette fused with smart city grid. Fiber-optic threads weave through cedar structural beams connecting circular community nodes. Deep navy background, cedar brown and electric cyan accents. Abstract — no specific nation's architectural details." --style indigenomics -m flash

# River flow path — values to outcomes
python generate.py -p "River-shaped flow diagram: 'Community Values' at source flowing through 'Design Principles' rapids, 'Capital Formation' bend, arriving at 'Sovereign Outcomes' delta. Cedar brown and teal on deep navy. Flowing organic paths not arrow boxes." --style indigenomics -m flash

# Data as territory
python generate.py -p "Canada territory map with glowing nodes marking Indigenous-led enterprises. Organic overlay of braided data streams connecting nodes. Deep navy, cyan glow, warm ochre land areas. Abstract — no nation flags or specific cultural symbols." --style indigenomics -m pro
```
