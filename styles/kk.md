# BC + AI Brand Style Guide

## Overview

The BC + AI brand style represents professional, modern editorial design with a dark, tech-forward aesthetic. This style is used for corporate content, thought leadership for the BC + AI ecosystem, and professional presentations.

## Color Palette

### Primary Colors
- **Dark Background Gradient**: `#0f0f1a` to `#1a1a2e`
  - Deep, rich blacks that create depth and sophistication
  - Gradient creates visual interest without overwhelming content

### Accent Colors
- **Teal**: `#00c8b4`
  - Primary brand accent for highlights, CTAs, and interactive elements
  - Conveys innovation, technology, and forward-thinking

- **Purple**: `#9333ea`
  - Secondary accent for emphasis and variety
  - Adds warmth and creativity to the tech aesthetic

### Supporting Colors
- **High Contrast**: White or near-white text (#ffffff, #f5f5f5)
- **Mid-tones**: Grays for supporting text and UI elements

## Visual Style

### Aesthetic
- Modern editorial illustration style
- Professional and polished
- Clean, crisp lines and shapes
- High visual impact with strong contrast

### Typography
- Sans-serif, modern typefaces
- Clear hierarchy
- Excellent readability on dark backgrounds

### Composition
- Strong focal points
- Balanced use of negative space
- Grid-based layouts where appropriate

## When to Use KK Style

Use the BC + AI brand style for:

- **Corporate Content**: Official ecosystem announcements, reports, presentations
- **Professional Thought Leadership**: BC + AI branded articles, talks, whitepapers
- **Marketing Materials**: Social media graphics, event promotions, pitch decks
- **Community Assets**: Meetup graphics, cohort materials, workshop assets
- **Technical Documentation**: Diagrams, architecture visuals, system maps

## When NOT to Use KK Style

Avoid this style for:
- Personal blog content (use HOPECODE instead)
- Anti-establishment or radical thought leadership (use HOPECODE)
- Organic, community-first narratives (use HOPECODE)
- Indigenous or decolonial content (consider cultural protocols)

## Implementation Notes

This style is automatically applied when using:
```bash
npx image-gen --prompt "your prompt"
# or explicitly
npx image-gen --prompt "your prompt" --style kk
```

The style suffix is defined in `styles/styles.yaml` and can be modified there without changing code.

## Design Philosophy

The BC + AI brand style balances:
- **Authority**: Professional quality that commands attention
- **Innovation**: Forward-thinking tech aesthetic
- **Accessibility**: High contrast for readability
- **Impact**: Visually striking without being overwhelming

It's designed for scale, consistency, and professional credibility across the ecosystem.
