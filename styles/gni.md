# GNI Cosmic Editorial Style Guide

## Overview

The `gni` style is the visual language of the Google News Initiative AI Journalism Lab: deep space-blue, orbital constellation diagrams, editorial precision at cosmic scale. It treats data as astronomy — swarms become signals, connections become constellations, complexity becomes something you can navigate like a star map.

This is editorial-meets-engineering. Science-fiction journalism, not flat corporate deck.

## Color Palette

- **Background:** Deep space-blue gradient — near-black to rich midnight blue
- **Accent glows:** Soft blue-white orbit trails, glowing node halos
- **Prism accents:** Split-spectrum for duality/contrast diagrams (bias splits, before/after)
- **No earth tones, no organic green, no mycelial aesthetic** — that's `hopecode`

## Key Visual Elements

- **Radial node layouts** — central hub with orbiting theme tags or sub-concepts
- **Orbit trails and constellation lines** — glowing dotted paths connecting nodes
- **Swarm-to-signal transformations** — dots clustering into patterns into story
- **Prism splits** — symmetrical duality compositions (chaos vs. clarity, raw vs. refined)
- **Gradient pulses** marking transformation points along logic flows
- **Clean sans-serif labels** with editorial hierarchy
- **Modular tile compositions** — cosmic dashboard layouts on galaxy-blue backgrounds

## Layout Patterns

- **Orbit Map** — central hub, orbiting tags
- **Prism Split** — symmetrical duality, two halves
- **Cosmic Dashboard** — modular tiles on galaxy background
- **Funnel** — question flow → AI filter → editorial action
- **Remix Engine** — inputs → AI core → output stream

## Tone

Playful precision. Not cold, not corporate. Science-fiction journalism: curious, editorial, technically grounded. Characters allowed (space miners, AI librarians). Concept over metaphor, but metaphor makes it land.

## When to Use

- GNI coaching session slides and diagrams
- AI-in-journalism explainer graphics
- Editorial tech diagrams (LLMs, inference pipelines, bias visualizations)
- Data storytelling where you want cosmic/constellation metaphors
- Professional journalism-AI workshop materials

## When NOT to Use

- Grassroots / community-first narratives (use `hopecode`)
- Punk/resistance/manifesto imagery (use `zine`)
- BC AI ecosystem branding (use `bcai` or `kk`)
- Anything organic, mycelial, or earth-toned

## Example CLI Command

```bash
python generate.py -p "AI journalism toolkit — four orbiting quadrants: GenAI, NLP, ML, Predictive Analytics — radial nodes on deep space-blue, glowing orbit connectors, clean sans-serif labels" --style gni -m pro
```

## Notes

**Pairing with kk:** `--style gni+kk` layers GNI cosmic aesthetics with BC+AI professional polish. Both specify backgrounds (space-blue vs. dark gradient), and later directives in the composed suffix tend to win — test both orders if results are unexpected.

**No mycelial bleed:** If you need to diagram a community network, resist the temptation to use `hopecode` conventions in GNI context. The `gni` style uses avatar node clusters and constellation lines, not organic branching paths. Keep it starfield, not forest floor.

**Prism splits for duality:** For bias diagrams, overrepresentation vs. output splits, or chaos vs. clarity comparisons — specify "symmetrical prism split" in the prompt body. The style suffix primes the model; the prompt directs the specific layout.

**Source material:** the GNI cosmic diagram style guide — principles, layout recipes, icon kit, and sample prompt snippets — is kept in a private knowledge base, not in the public tool repo.
