# Keynote Visual Workflow Prompt Pack

Public use case: [AI Keynote Slides Need Taste Before Prompts](https://kriskrug.co/2026/06/04/ai-keynote-slides-visual-workflow/)

This small prompt pack is designed for dry-run onboarding and product design.
It shows how a keynote visual workflow can move from source idea to reviewable
graphic candidates without including private references or rejected batches.

## 1. Artifact Chain Hero
**For:** Blog hero or guide opener
**Aspect Ratio:** 16:9
**Style:** hopecode
**Manual Notes:** Leave room for real text to be rebuilt in a slide or article layout.
**Prompt:**
> A diagrammatic editorial hero showing one idea moving through a creative pipeline: raw talk notes, reference board, prompt pack, generated image candidates, review selector, slide canvas, and downstream website or social assets. The flow should feel hand-built, organic, and taste-led rather than corporate. Use textured paper, marginal notes, visible decision gates, small contact-sheet thumbnails, and no readable brand logos or fake UI text.

## 2. Reference Kit Before Generation
**For:** Style chooser and reference-first workflow example
**Aspect Ratio:** 16:9
**Style:** hopecode
**Manual Notes:** Useful for explaining why style choice comes before batch generation.
**Prompt:**
> A visual metaphor for a reference kit assembled before AI image generation: mood scraps, color swatches, field-note annotations, style cards, and a few blank image placeholders arranged on a working table. The image should communicate taste, constraints, and visual direction before any model output exists. Avoid generic glowing AI motifs, dashboard screens, robots, and polished stock-photo aesthetics.

## 3. Review Gate And Anti-Slop Selector
**For:** Library/review workflow example
**Aspect Ratio:** 1:1
**Style:** hopecode
**Manual Notes:** Use as a product prompt for approval-state UI thinking.
**Prompt:**
> A square editorial illustration of a review gate for generated images: a contact sheet of candidate graphics, some marked reject, some marked regenerate, one marked approved, and one marked rebuild manually. The scene should emphasize human judgment, provenance, and taste. Use analog markup, stickers, crossed-out thumbnails, and subtle archive labels, but do not render readable interface text or brand logos.

## 4. Slide To Downstream Artifacts
**For:** Artifact-chain library example
**Aspect Ratio:** 16:9
**Style:** hopecode
**Manual Notes:** Shows the image-to-slide-to-website reuse path.
**Prompt:**
> A layered creative workflow scene showing a keynote slide asset branching into a blog post image, guide diagram, social card, speaker asset, and website section. The source graphic stays visible at the center while downstream formats fan outward like handmade production proofs. The composition should make reuse and provenance clear. Keep all text areas abstract or blank for manual layout later.
