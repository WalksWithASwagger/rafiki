# ED + AI Round 8 - Futureproof Mythic Education

**Goal:** Create ten completely new `ED + AI` / `AI Education Meetup` logo concepts that borrow the strongest Futureproof Festival energy: luminous dot-matrix signal, Pacific Northwest night colour, handmade future-folk artifacts, portals, talismans, seed vaults, cabinets of wonders, and living technology. This round should feel weirder, braver, and more memorable than the corporate/logo-factory attempts.

**Creative frame:** Think of `ED + AI` as a Futureproof-adjacent education sub-brand inside the BC + AI ecosystem. It should feel like a secret night-school sign, a teaching talisman, a threshold marker, or a community learning artifact. Use the Futureproof reference materials as a taste board only. Do not copy `FUTUREPROOF` text, event marks, dates, or any prior generated `ED + AI` image.

**BC + AI relationship:** The official BC + AI logo is presenter / ecosystem context, not the main visual engine. `ED + AI` must be the hero. Add a small `BC + AI` parent mark, endorsement, or lockup only when the prompt asks for it.

**Text policy:** Primary text must be `ED + AI`. Optional secondary text: `AI Education Meetup`. Optional tiny parent credit: `BC + AI`. Generated text is rough concept evidence only; final typography will be manually rebuilt.

**Style pressure:** Mythic, pop, luminous, handmade, artifact-like, educational, usable as a logo seed. More Futureproof Festival night signal than Hope Code grit. Less corporate. Less credential badge. Less SaaS. Less generic Afrofuturism. More strange local learning magic.

**Hard bans for every prompt:** corporate tech logo, generic conference branding, SaaS gradient, fake institutional seal, sponsor-deck polish, graduation cap, apple, school bus, lightbulb, robot, robot teacher, generic glowing brain, dashboard, neural network wallpaper, hex-grid tech background, crypto/web3 gloss, copied Futureproof wordmark, copied Ethos marks, copied cultural symbols, kente, adinkra, masks, faces, flags, invented extra sponsors, tiny fake unreadable body copy.
