# BC+AI Communities and Subgroup Graphics Prompt Kit

**Goal:** Generate candidate graphics for the BC+AI Communities page and its subgroup cards. The main Communities hero is the true asset gap. Subgroup graphics are optional replacements only: they must clearly beat the current live assets before being swapped in.

**Output root:** `output/bcai-communities-subgroups/`

**Review rule:** Compare every candidate against the current live page asset in the same slot. Approve only if it is more specific to the subgroup, more legible in a website crop, and more clearly BC+AI than the live asset. Do not approve generic AI, generic meetup, or generic Vancouver stock-feeling images.

**Text rule:** No baked-in headline, title, subgroup name, event copy, dates, URLs, logos, or readable signage. Leave clean space for real website copy and overlays.

**Shared negative constraints:** no glossy SaaS dashboards, no corporate stock illustration, no neon circuit board cliches, no robot hands, no floating chatbot icons, no unreadable generated text, no fake logos, no crypto/web3 vibes, no sterile conference-stage imagery, no single-person hero portrait unless explicitly requested later.

---

Website hero crop, premium community artifact feel, organic network structure, no readable text or baked-in copy.
