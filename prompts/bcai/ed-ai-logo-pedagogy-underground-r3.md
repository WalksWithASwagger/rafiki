# ED + AI Round 3 - Pedagogy Underground Logo Sprint

**Goal:** Explore a real identity for `ED + AI`, not a decorative child logo. The visual thesis is **Pedagogy Underground**: education as hidden care labor, consent, repair, ungrading, commons, culture-based STEAM, youth agency, and relational infrastructure.

**Ethos / research posture:** Ethọ́s Lab influence is an abstract homage only. Draw from Ubuntu as reciprocal structure, Afrofuturist youth worldbuilding, culture-based STEAM, and storyteller warmth. Do not copy Ethọ́s Lab marks, write the Ethọ́s name, or borrow cultural symbols.

**Parent brand posture:** The attached BC + AI ecosystem logo is a small presenter / parent-system reference and a plus-geometry reference. It must not become the hero. Use `BC + AI` only as a tiny credit if it helps the layout.

**Text policy:** Rough generated text is acceptable for concept exploration. Use only `ED + AI`, optional `AI Education Meetup`, optional tiny `BC + AI`, and short one-word concept annotations only where a prompt asks for them.

**Hard bans for every prompt:** robots, robot teachers, glowing brains, dashboards, holographic UI, graduation caps, apples, school buses, lightbulbs, generic node networks, hex grids, SaaS gradients, fake institutional seals, copied Ethọ́s marks, the visible words `Ethos`, `Ethọ́s`, `Atlanthos`, `Ubuntu`, or `Griot`, borrowed cultural symbols, masks, faces, sacred geometry, and fake sponsor/event details.
