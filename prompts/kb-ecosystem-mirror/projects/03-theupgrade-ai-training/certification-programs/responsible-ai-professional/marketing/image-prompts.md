## 1. Luma Event Banner

**Specs:** 1920 x 1080 px
**Purpose:** Header image for Luma event page

### Prompt Option A (Abstract/Conceptual)

```
Abstract visualization of AI ethics and human judgment. Neural network patterns intersecting with human elements. Color palette: deep teal, warm amber, white. Clean, sophisticated, not dystopian. Subtle grid patterns suggesting frameworks and structure. Modern tech aesthetic with warmth. No faces, no robots. Professional certification feel. --ar 16:9 --style raw
```

### Prompt Option B (Vancouver-Inspired)

```
Aerial view of Vancouver waterfront at golden hour, overlaid with subtle geometric patterns representing AI frameworks. Mountains in background. Color palette: pacific blues, forest greens, warm gold accents. Modern, hopeful, grounded in place. Subtle tech elements integrated naturally. Professional, clean, not busy. --ar 16:9 --style raw
```

### Prompt Option C (Minimal/Typography-Forward)

```
Minimalist abstract background with subtle gradient from deep navy to teal. Geometric shapes suggesting balance and ethics - scales, interconnected nodes, human silhouettes abstracted into simple forms. Negative space for text overlay. Professional, modern, trustworthy. --ar 16:9 --style raw
```

## Typography Recommendations

**Headlines:** Inter, Outfit, or similar modern sans-serif
**Body:** Inter, Source Sans Pro
**Accent:** Consider a slightly condensed weight for "RAP" wordmark

---

## Production Notes

1. **Generate base images** using prompts above
2. **Add text overlays** in Canva, Figma, or preferred design tool
3. **Export at 2x resolution** for retina displays
4. **Test social previews** before posting (use LinkedIn post inspector)
5. **Keep originals** in project folder for future cohorts
