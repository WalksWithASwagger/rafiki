# Show shirts — five Vancouver AI graphics (dark-on-dark, no robot/keyboard)

Hardcore lot-poster energy for **real dark tie-dye blanks** (navy/black/magenta/teal). Flat print art — not a photo mockup.

**Batch:**

```bash
cd tools/image-gen
npx image-gen ../../content/projects/apparel-screen-print/image-prompts-show-shirts-five-dark-drippy.md \
  --reference-images "../../content/projects/apparel-screen-print/refs/ref-user-finalist-circular-vortex-vancouver-ai.png,../../content/projects/apparel-screen-print/refs/ref-user-finalist-bone-shield-sunburst-vancouver-ai.png,../../content/projects/apparel-screen-print/refs/ref-user-finalist-circular-vortex-vancouver-ai.png,../../content/projects/apparel-screen-print/refs/ref-user-finalist-bone-shield-sunburst-vancouver-ai.png,../../content/projects/apparel-screen-print/refs/ref-user-finalist-circular-vortex-vancouver-ai.png" \
  --model gemini-3-pro-image-preview \
  --resolution 2K \
  --aspect-ratio square \
  --output-dir ../../content/projects/apparel-screen-print/outputs/show-shirts-2026 \
  --no-style
```

**Banned everywhere:** robot, android, keyboard, synthesizer, DJ deck, “cute” mascot, corporate logo grid, stock vector gloss, airbrushed plastic.

## 1. Show print — Acid autopsy melt

**For:** Chest hit on black/navy dye; needs neon edge glow to read

**Prompt:**
> Original Vancouver AI **show T-shirt graphic** — go over the top. Background: near-black void (#050508) with subtle deep purple haze. Massive custom type: VANCOUVER arched top, AI huge center, COMMUNITY on a dissolving ribbon — letters **melting into long acid-green and toxic-yellow slime drips** that pool at the bottom. Add **misregistered CMYK ghost** edges (cyan/magenta offset) on key strokes. Center: ribcage silhouette opening into a **green-black nebula** with tiny stars. Red roses with thorns climbing the type. Heavy black outer strokes, inner halftone stipple, gritty screen-ink texture. **Dark-on-dark** but punctured by **violent neon** so it reads from stage distance. Hand-drawn, wrong, iconic. No robots or musical instruments.
