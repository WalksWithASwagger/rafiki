## 1. Rally metaphors (single-scene)

**Hook (copy):** *The future of Vancouver's tech scene isn't in the WebSummit schedule — it's hiding in the margins, behind badges, beneath buzzwords.*

```
A dimly lit tech conference, faces in neon, overhead screens with corporate slogans; in the shadows a hand-drawn map, graffiti between badge lanyards, zines on the floor; corner hack circle, rebels rewriting the code. West coast zine, analog grain, high contrast, no stock photo.
```

**Frame the tension:** *WebSummit is both a gateway and a game…*

```
Massive glowing gateway over a cybernetic city, algorithmic sentinels; one side white-lit startup booths, other side glitch-cloaked figures at a cracked console labeled “Root Access: Community Protocols”; binary like scripture in the sky. Diagrammatic, flat layers, dark teal and copper.
```

**Thesis:** *We're not here to assimilate — we're here to subvert…*

```
Figure in moss jacket hijacking a satellite dish on a brutalist roof, rerouting signal to a community garden with kids on reclaimed laptops; glitch in skyline reveals green city beneath. No text in image. Solarpunk cyberpunk hybrid.
```

**Identity:** *Solarpunk syntax and cyberpunk intent…*

```
Hybrid tree from circuitboard soil, fiber-optic branches, QR leaves; deconstructed datacenter rewilded with vines; Indigenous coder etching new glyphs into bark. Respectful, non-appropriative, symbolic not documentary.
```

**Network visual:** *Neural forest…*

```
Mycelial neural paths under a stylized Vancouver skyline; nodes as zine collectives, toolkits, healing circles; glass towers with empty pitch decks above, living network below. Isometric, hand-inked, bioluminescent accents.
```

**Tool of tools:** *The app is not the map…*

```
Broken smartphone UI overlaid on hand-drawn topo map of sticky notes and Polaroids; grayscale UX vs full-color messy margin notes. Exploded diagram, editorial illustration.
```

**Resistance visual:** *Startups are survival tools if wielded wisely…*

```
Pitch deck in ceremonial fire; circle of people on typewriters, tablets, chalkboards; drone with cracked lens, feed intercepted by pirate signal. Zine cover composition.
```

**Transmission ending:** *WebSummit is a satellite — we're building ground stations…*

```
Large WebSummit broadcast satellite, sanitized PR beam; foreground DIY antenna from bike parts, solar, driftwood, glowing alternate spectrum. Wide cinematic, no readable logos.
```

**Wild card:** *We're not here to scale. We're here to sync.*

```
Two people in kelp forest, DIY neural headsets, eyes closed, syncing; “datafish” of light, bioluminescent glyphs. Dreamlike, not kitsch, Pacific northwest underwater palette.
```

## 2. Behind the Terminal (section diagram prompts)

**§ System scan — what’s really happening**

```
Split screen: half sterile conference with logo walls; half hand-drawn mindmap of glitched faces, tangled networks; text fragments as abstract shapes only: “visibility ≠ legitimacy”, “ecosystem or echo chamber?” — solarpunk meets cyberpunk, layered textures, transparent overlays, diagram style.
```

**§ Code your crew — counter-network**

```
Decentralized starburst: nodes as people and places (warehouse meetups, forest coding, rooftop); lines glow with mutual aid not metrics; “offline trust zones” hand annotation; isometric, warm, biopunk x community zine.
```

**§ Exploit the platform (without being owned)**

```
Tactical overlay: silhouettes “hacking” app interface; red string to real map pins; icons: eye, QR, rooftop router; blueprint meets stencil-punk, solarpunk twilight palette.
```

**§ Insert systemic critique — pitch as protest**

```
Radial diagram: press release text on outer ring, peeling to inner rings: Land, Power, Story, Right to speak; tree rings, Indigenous pattern hints as *abstract* geometry, climate chart hints, mixed media collage, ink + glitch.
```

**§ Generate your own legitimacy**

```
Collapsing legacy media tower (cracked glass, generic logos); beside it “forest” of signal-trees: Zine, Discord, co-op stream; hand-drawn trust paths, solarpunk topography, paper map + light leak.
```

**§ Weaponize authenticity**

```
Exploded diagram of human silhouette: brain as poetry, heart as code, hands as ritual, voice as waveform; mics and cameras as labeled pressure; lightning path “raw signal”. Anatomical zine + analog glitch.
```

**§ Rebel loop — night summits / off-script**

```
Vancouver at night, circular map overlay; “anomalous” lit zones (rooftops, alleys, ocean); arcs like lunar calendar; tarot-style glyphs for sync, glitch, dream, signal return; dream logic + tactical cartography.
```

**Full article / cover diagram**

```
Poster network: mossy server, burning press badge, rooftop code ritual, cracked WebSummit banner; hand-drawn arrows, punk glyphs, solarpunk flowers from fiber roots; title area blank or placeholder. Monospace graffiti font zone.
```

## 3. Ten ecosystem-skeptic resistance diagrams

1. **Startup theatre feedback loop** — Satirical flowchart: Pitch → Panel → PR hit → Funding round; nodes detach from “community outcome”; margin notes “real innovation?”, “extractive hype spiral”; infovis + zine collage, duct tape, ironic corporate glaze glitching.

2. **Extractive infrastructure blueprint** — Blueprint: “conference pipeline” community energy in → marketing out; arrows: sponsorship, LinkedIn, metrics; hidden pipes: attention, IP, local credibility; tech dystopia × drafting table.

3. **Regenerative tech stack** — Vertical stack: Earth / Community / Code / Application; tree roots; red annotations where extractive models cut corners. Solarpunk, instructional poster.

4. **Colonial software patterns** — Snake loop “Colonial Pattern 1.0”: steps as abstract pipeline; alternate arrows: reconnection, mutual aid, co-ops, circular investment. System diagram, uncomfortable clarity.

5. **The LARP of innovation** — Two-act theatre diagram: Act I pitch/badges; Act II backstage burnout/sidelining; protest-poster schematic.

6. **Decentralized org chart of real power** — Traditional org chart dissolving into spiral; new chart: elders, artists, organizers, hackers; trust lines, fractal geometry. Not portraits—roles as silhouettes.

7. **Signal vs spectacle map** — 2×2: Impact/Optics × Local/Global; quadrant examples as abstract labels only. Crisp diagram, readable at slide distance.

8. **DIY anti-conference kit** — RPG character sheet: tools (zines, DMs, barter, lore); skills; inventory (QR, chalk, cards); mossy street corner. Field manual look.

9. **Firewall for the local commons** — Hex grid firewall around ecosystem; policy nodes, review boards, open zones; incoming arrows “bouncing”; analog glitch, readable legend.

10. **Ecosystem time spiral** — Spiral: seasonal (seed, root, bloom, compost) vs jagged “hype cycle”; label “where growth ≠ extraction”. Time diagram, two tempo curves.

## 7. West Coast VanAI diagram vocabulary

**Vibe:** solarpunk, Pacific NW, DIY, zine glyph, mycelial network, temporary autonomous clusters.

**Symbol cheat sheet (for labels / icons in diagrams)**

| Glyph idea | Use for |
|------------|--------|
| Signal spiral | Ideas propagating |
| Root node | Values, start of dialogue |
| Decentralized starburst | Community network |
| Temporary autonomous cluster (TAC) | Meetup / event node |
| Protocol glyphs | Tools, rituals, shared practice |
| Solar vector grid | Time + intention |
| Mirror arc | Feedback loops |

**Layout recipes (names only — use as `::diagram` tags in full prompts)**

- Hopecore nexus map (radial, north star center, vibe clusters)  
- Solarpunk stacks (culture / tech / energy / imagination layers)  
- Fractal loop map (Möbius, reflection arcs)  
- City as canvas grid (Vancouver as motherboard + constellation)  

**Label vocabulary (short phrases for on-image text)**

Neural Grove · Protocol Jam · Sunbeam Node · Commons Stack · Care Signal · Techno-sapien · Civic Bioluminescence · Rewilded Code · Ancestral algorithm *(use with care; avoid appropriation in visuals)*

**Style tags (append to image prompts)**

Hand-drawn lines, layered transparency, earth + iridescent neon, paper grain, water-ink bleed, circuit overlay, monospace + handwritten margin notes, **no** generic stock conference photo.
