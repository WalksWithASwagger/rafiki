# Blue Engine Collaborative — diagram style crib

*Secondary reference: “Blue Engine” optimistic coaching / transformation aesthetic when you need a **clean civic deck** look instead of HOPECODE grit. This is **not** KK default visual language; use for client-matched or comparison decks.*

## Positioning (from public site pattern)

- Hands-on, optimistic, mission-driven org acceleration.
- Verbs: accelerate, test, learn, scale.
- **Feel:** practical, confidence-building, light—not academic or grim.

## Sample diagram shape (transformation flywheel)

- Stages: Insight → Rapid pilot & test → Measure & learn fast → Scale (with curved arrows, not harsh zigzags).
- **Flow over boxes**; gentle curves echo “wave” hero graphics.

## Color (example tokens from crib)

| Token | Use |
|-------|-----|
| Primary Cobalt `#0052CC` | Headings, outer strokes, anchors |
| Bright Sky `#36B3FF` | Highlights, gradient stops |
| Navy Ink `#06233F` | Body metrics, $ / % callouts |
| Slate-50 `#F5F7FA` | Neutrals |
| Optional teal pop `#3FE7D9` | Success / growth only |

- Rule of thumb: **~70%** neutral, **~20%** cobalt, **~10%** sky accent.

## Typography (crib)

- Headers: Montserrat or Poppins bold; moderate tracking.
- Body: Inter or Source Sans; generous line height.
- All-caps only for small micro-labels (≤12px).

## Shapes and lines

- Rounded rectangles (~4px radius), **2px** strokes, soft **wave** connectors.
- Faint background wave: linear gradient (e.g. light blues at low opacity) behind key nodes.
- **Coach’s highlight:** subtle sticky-note or tooltip blocks (light cobalt fill) for action steps.

## Iconography

- Simple **stroke** icons, 2px, rounded joins; match diagram corner radius.

## Tone

- Upbeat, pragmatic; one key idea per frame; **large** padding (40–60px feel).

## Accessibility

- WCAG AA: cobalt on white and white on cobalt; check gradient legibility.
- QA pass in **grayscale** for hierarchy before final color.

## Contrast with HOPECODE (KK / GNI ethical diagrams)

- Blue Engine: **flat**, **predictable** curves, **hex-led** palette, coaching clarity.
- HOPECODE: **jittered** linework, **found** texture, **no** fixed hex, root-first, anti–spreadsheet.

Use this file when someone asks for “Blue Engine style”; use `hopecode-style-guide.txt` for default diagram generation in this program.
