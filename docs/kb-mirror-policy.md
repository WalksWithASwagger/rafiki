# KB Mirror Policy

**Rafiki** (`prompts/kk-kb/`) is the authoritative home for KK's image and
diagram prompt files. These assets were migrated from the `kk-ai-ecosystem`
knowledge base on 2026-04-25.

Repo: <https://github.com/WalksWithASwagger/rafiki>

## Rule

If you need to update one of these prompt files, edit the Rafiki version. Do
not edit copies that may still exist in `kk-ai-ecosystem/content/reference/` —
those are pre-migration mirrors and any changes there will be lost or create a
silent divergence.

The stub-replacement work in the `kk-ai-ecosystem` repo is owned by that
repo's contributors, not Rafiki.
